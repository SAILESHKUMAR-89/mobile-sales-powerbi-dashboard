# 📊 Mobile Sales Dashboard — Power BI Project

## Overview

This project presents an interactive **Mobile Sales Analytics Dashboard** built using **Power BI**.  
The dashboard analyzes mobile sales performance across cities, mobile models, payment methods, and customer ratings.

The goal of this project was to explore how a **complete analytics dashboard can be built entirely within Power BI**, including data preparation, KPI creation, and visualization.

---

## Key Metrics

- **Total Sales:** 769M
- **Total Quantity Sold:** 19K
- **Total Transactions:** 4K
- **Average Sales Value:** 40.1K

---

## Visualizations Used

### KPI Cards
Displays overall business performance metrics.

### Monthly Sales Trend
Line chart showing quantity sold across months.

### Mobile Model & Brand Performance
Column chart comparing sales across different mobile models and brands.

### Payment Method Distribution
Donut chart analyzing customer payment preferences.

### City-Level Sales Map
Geographical visualization of total sales by city.

### Customer Rating Analysis
Bar chart showing distribution of customer satisfaction ratings.

### Product Summary Table
Detailed breakdown of sales, quantity, and transactions by mobile model.

- Month filter
